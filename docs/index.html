Introduction

Single-cell transcriptome sequencing (sc-RNA-seq) experiments allow us to discover new cell types and help us understand how they arise in development. The Monocle 3 package provides a toolkit for analyzing single-cell gene expression experiments.

References

Documentation: Trapnell lab github documentation

Tool Description

Input Files

  • This module accepts either a zipped 10x Genomics file containing the counts and experiment row and column metadata or a Serurat SingleCellExperiment object in the R-language RDS or Robj formats.
  •  

Output Files

  • <output_filename>_monocle.robji
    binary serialize R object containing the Monocle SingleCellExperiment object.
  • <output_filename>_.pdf
    Plots of the datasetcolored by the parameters specified in the <plot_colorby> parameter with the branch graph overlayed prior to generating pseudotime mapping.
  • <output_filename>_.pdf
    Plots of the data colored by the parameters specified in the <plot_post_colorby> parameter with the pseudo-time trajecotry overlayed.

Parameters

Inputs and Outputs

Name Description
input seurat rds file* A Serurat SingleCellExperiment object in the R-language RDS or Robj formats.
output filename Prefix used for the name of all generated output files.
input 10x file A 10x Genomics xipped data file containing counts and experiment row/col metadata.
root cells Root cells to identify initial state for pseudo-time ordering.
root nodes Root nodes to identify initial state for pseudo-time ordering.
plot colorby Phenotypic (column) information to color samples by in plots prior to pseudo-time ordering. Phenotype labels should be comma-delimited and must exist in the input dataset.
plot post colorby Phenotypic (column) information to color samples by in plots after pseudo-time ordering is complete. A pseudo-time colored plot is always produced and does not need to be specified. Phenotype labels should be comma-delimited and must exist in the input dataset.
